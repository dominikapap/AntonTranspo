## General info
This project is a webpage for a fake bus transport company Anton Transpo, you can check some general info, send a message using prepared form(created using Python) or calculate a ride cost(working with Javascript). Project is currently live at http://dominikapapierska.pythonanywhere.com/


## Technologies
Project is created with:
* Python 3.9.7
* Django 3.2.5
* HTML5
* CSS
* Bootstrap 5
* Javascript

## Setup
* install packages from requirements
* change SECRET_KEY in settings
* add your own gmail address and password
